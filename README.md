This Power BI dashboard analyzes customer behavior and product performance using raw data scraped from the Walmart website. The data pipeline includes data collection, cleaning, transformation, and modeling, resulting in a structured analytics-ready dataset. The dashboard incorporates customer reviews, ratings, product attributes, and seller information to uncover trends that impact purchasing decisions and overall performance.

The dashboard is organized into three core analytical views:

Overview

Provides a high-level snapshot of overall marketplace performance, including key metrics such as average ratings, recommendation percentages, review volume, and sales-related indicators. This page helps quickly identify broad trends, outliers, and areas requiring deeper investigation.

Product Insights

Focuses on individual product performance by analyzing ratings, review sentiment indicators, keyword frequency, and customer feedback patterns. This view highlights which product features drive satisfaction or dissatisfaction and identifies common issues reported by customers.

Vendor Insights

Evaluates seller-level performance, comparing review volume, customer sentiment, and product quality indicators across vendors. This page helps identify top-performing sellers, vendors with recurring customer complaints, and opportunities for operational or quality improvements.

Overall, this dashboard demonstrates an end-to-end analytics workflow, transforming unstructured web data into actionable insights that support data-driven decision-making around product optimization, customer experience, and vendor performance.
